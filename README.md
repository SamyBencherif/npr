## installation

```bash
curl https://raw.githubusercontent.com/SamyBencherif/npr/refs/heads/main/npr.py > ~/.local/bin/npr && chmod +x ~/.local/bin/npr
```

### installation prerequisites

For this program to work the following must be true.

1. Python3 is installed
2. There is a `~/.local/bin` folder and it is in the PATH
3. The OS is UNIX-like (For Windows users you may use something like git bash or invoke the script via python, `py npr`)

## usage

Use the newly installed "npr" command.
```bash
npr
```

Press Q to navigate out of an article or quit the program.
